# PokeFinder
PokéFinder is a Pokédex implementation made in Vanilla JavaScript with SASS Styling.

It uses the [PokéAPI](https://pokeapi.co/).

![PokéFinder screenshot](public/media/preview.png)

# Quick Guide

## Starting the live server
```js
npm run start
```

## Compiling the SASS

```js
npm run style 
```
